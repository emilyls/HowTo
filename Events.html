Events

It is possible to request information regarding events happening at different Oregon State parks. The request returns a description of the event, the park name and id, the start and end dates and times, where at the park the event is located, the event title and id, an event category and whether it is an all day event. This request can be used to create a calendar of events or to provide information about what is happening at a specific park.

Requests for event data are sent to: 

http://oregonstateparks.org/data/index.cfm/parkEvents?
With the following possible parameters: 
parkId=
categoryId=
descr=
dateFrom=
dateTo=
eventId=
The following are the current category IDs which can be used: 
0 = all
1 = General
2 = For Kids
3 = Nature
4 = State Parks Day

Multiple words can be used to search the event description and a like search is used. Each keyword must be sent as a new descr=keyword pair: 

http://oregonstateparks.org/data/index.cfm/parkEvents?descr=food&descr=fun
Dates must be sent in the format MM/DD/YYYY and dateFrom returns all dates including and after the date provided, while dateTo returns all dates before and not including the date provided. 
http://oregonstateparks.org/data/index.cfm/parkEvents?dateTo=12/15/2015
Multiple parameters can be sent as well: 
http://oregonstateparks.org/data/index.cfm/parkEvents?parkId=103&dateTo=12/15/2015
The default search will return all events beginning the day the request is made.

The HTTP request can be constructed in the same way as previous requests, once you have a constructed a string with the desired parameters:

var options = {
  host: 'oregonstateparks.org',
  path: '/data/index.cfm/parkEvents?' + string 
};
