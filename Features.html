Features

The most basic call to the Oregon State Parks API, simply returns a list of all of the feature categories and titles at the parks. If you are looking at the Oregon State Parks website, these are the little icons which appear on each park's page. No parameters can be used, a simple GET request is sent to: 

http://oregonstateparks.org/data/index.cfm/features?

The HTTP request can be constructed in the same way as the first request for Park data:

var options = {
  host: 'oregonstateparks.org',
  path: '/data/index.cfm/features' 
};

Format of JSON returned by request:

[
  {
    "featureClass":"ampitheater",
      "featureTitle":"Ampitheater"
  },
  {
    "featureClass":"beach",
    "featureTitle":"Beach Access"
  },
  {
    "featureClass":"bike",
    "featureTitle":"Bike Path"
  },

   // More Features
