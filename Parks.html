Parks

The most general call that can be made to the Oregon State Parks API results in a list of all of the state parks in Oregon. The GET request is sent to:

http://oregonstateparks.org/data/index.cfm/parks

If you are using node.js, you can set this up with an HTTP request. The data will be returned in chunks, so it is important to concatenate those before attempting to parse the JSON. Once you have the data stored as a JSON object, you can pull out the data you are interested in using.

var options = {
  host: 'oregonstateparks.org',
  path: '/data/index.cfm/parks'
};
callback = function(response) {
  var string = '';
  response.on('data', function (chunk) {
    string += chunk;
  });
  response.on('end', function() {
    var data = JSON.parse(string);
    // Your code here
  });
}
http.request(options,callback).end();

This is a good request to use if you want a list of park names and ids, since most of the other requests that can be used in the API can use this information to return more specific information. You cannot include any parameters in the GET request, so it will always return all of the parks.
